CHM反编译html脚本

2012-12-10

借助windows自带命令行工具hh.exe。

@echo off
title CHM电子书反编器BAT版
color a
echo.
set /p urlfile=请把要反编的CHM电子书拖进来(再按回车键)：
copy %urlfile% chmfile.chm > nul
hh -decompile .\html chmfile.chm
del /q chmfile.chm > nul 
echo.
echo 反编文件成功。保存在.\html文件夹中，按任意键退出！
rem pause > nul
exit
